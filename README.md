# Portfólio Profissional - Maria Isabella Gomes

Site institucional desenvolvido para a advogada **Maria Isabella Gomes**, com foco em presença presença digital profissional, autoridade jurídica e experiência do usuário moderna.

O projeto foi pensado para ser elegante, responsivo e alinhado às boas práticas éticas da advocacia, seguindo um visual atual (2026).

## 🧠 Objetivo do Projeto

Criar um site profissional para advocacia que:
- Transmita credibilidade e confiança
- Apresente informações de forma clara e institucional
- Facilite o contato com potenciais clientes
- Seja totalmente responsivo (mobile-first)

## 🛠️ Tecnologias Utilizadas

- **HTML5** – Estrutura semântica
- **CSS3** – Layout moderno, responsivo e animações suaves
- **JavaScript (Vanilla)** – Interações e melhorias de UX
- **Git & GitHub** – Controle de versão

---

## ⚙️ Funcionalidades

- Menu mobile com botão hambúrguer
- Scroll suave entre seções
- Header com efeito ao rolar a página
- CTA estratégico com redirecionamento para WhatsApp
- Animações leves ao entrar na tela
- Layout totalmente responsivo
